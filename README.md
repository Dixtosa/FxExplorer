<h1>ფუნქციები</h1>
ფუნქციების სიღრმისეული გამოკლვევა როგორებიცაა: 
* წარმოებული
* ფუნქციის მხების ფუნქცია
* დახრილი და ვერტიკალური ასიმპტოტა
* და სხვა.


აქვს zoom-ის საშუალება და დახაზული ფუნქციის სქროლით გადათვალიერების საშუალება.
